## Tools needed to run
1) NodeJS
2) NPM (will be set up with nodejs)
3) npm install node
4) npm install mysql2
5) npm install cors
5) npm install express
6) npm install @react-google-maps/api

To install all at once: 
npm install --save node mysql2 cors express @react-google-maps/api

## Starting the react server
npm run dev

## Starting the backend server
cd ./Backend
npm start

## Opening the live view
While the react server is running type o + enter in the console
